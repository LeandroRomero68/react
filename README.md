# EduTech

EduTech es una plataforma que ofrece cursos **online, presenciales e híbridos**, con funcionalidades de **perfil de alumno** y **administración de compras**.

## Temas implementados
- Registro de usuarios
- Inicio de sesión
- Listado de cursos
- Compra de cursos
- Detalle de cursos
- Editar cursos
- Eliminar cursos
- Perfil del alumno
- Validaciones de formulario
- Administración de compras
- Detalle de compra
- Editar compra
- Eliminar compra

---

## Tecnologías utilizadas
- React
- React Router (modo DATA)
- TailwindCSS
- JWT (Json Web Token)
